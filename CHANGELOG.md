# Changelog

All notable changes to this project are documented here. Releases are generated
with Commitizen from Conventional Commit history.

## 0.1.0

- Initial public release.
- Added `pai run` for structured runtime tracing.
- Added `pai bundle` for agent-consumable run summaries.
- Added events for run lifecycle, exceptions, imports, calls, pytest outcomes,
  and side-effect families.
