# Security Policy

PAI is experimental agent tooling. Do not use it for sensitive production
systems unless you have reviewed the collection behavior for your environment.

## Reporting A Vulnerability

Please report security issues privately by emailing:

```text
oriesh@icloud.com
```

Include:

- Affected version or commit.
- Steps to reproduce.
- Impact.
- Any suggested fix or mitigation.

## Data Handling

PAI is designed to record schemas and metadata, not raw user values. A report
about accidental value capture is a security issue.
